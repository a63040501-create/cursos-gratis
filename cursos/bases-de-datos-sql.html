Curso: Introducción a las Bases de Datos y SQL

Lección 1: ¿Qué es una Base de Datos?

Una **base de datos** es un sistema organizado para almacenar, gestionar y recuperar información de manera eficiente. Piensa en ella como una versión digital y extremadamente poderosa de una hoja de cálculo, capaz de manejar millones de registros y relaciones complejas.

Existen diferentes tipos de bases de datos. En este curso nos enfocaremos en las **bases de datos relacionales**, que organizan la información en tablas con filas y columnas, y utilizan **SQL** para comunicarse con ellas.

Lección 2: Fundamentos de SQL

**SQL (Structured Query Language)** es el lenguaje estándar para gestionar bases de datos relacionales. Es un lenguaje declarativo, lo que significa que le dices a la base de datos lo que quieres, en lugar de decirle cómo hacerlo.

Aquí tienes los cuatro comandos más importantes, conocidos como **CRUD** (Crear, Leer, Actualizar, Borrar): 


                -- LECTURA (READ): Selecciona datos de una tabla
                SELECT nombre, email FROM usuarios;

                -- CREACIÓN (CREATE): Inserta un nuevo registro
                INSERT INTO usuarios (nombre, email) VALUES ('Juan', 'juan@email.com');

                -- ACTUALIZACIÓN (UPDATE): Modifica un registro existente
                UPDATE usuarios SET email = 'nuevo@email.com' WHERE nombre = 'Juan';

                -- BORRADO (DELETE): Elimina un registro
                DELETE FROM usuarios WHERE nombre = 'Juan';

Lección 3: Conectando Python a una Base de Datos

Para usar bases de datos en tus aplicaciones de Python, necesitas un conector que permita a tu código comunicarse con la base de datos.

Te enseñaremos a usar librerías como `sqlite3` (que viene con Python) para ejecutar consultas SQL directamente desde tu código, lo que te permitirá crear aplicaciones web que almacenan y manipulan datos de forma persistente. 


                import sqlite3

                # Conecta a la base de datos (o la crea si no existe)
                conexion = sqlite3.connect('mi_base_de_datos.db')
                cursor = conexion.cursor()

                # Ejecuta un comando SQL
                cursor.execute('SELECT * FROM usuarios')
                
                # Cierra la conexión
                conexion.close()
