Curso: Desarrollo Front-end con React

Lección 1: ¿Qué es un Framework Front-end?

A diferencia de las páginas web tradicionales que recargan toda la página con cada clic, los frameworks modernos como **React** se usan para crear **aplicaciones de una sola página (SPA)**. En una SPA, el contenido cambia de forma dinámica sin necesidad de recargar la página, lo que proporciona una experiencia de usuario fluida y rápida. React se enfoca en la interfaz de usuario, permitiéndote construir aplicaciones con componentes reutilizables.

**React** es una librería de JavaScript de código abierto, desarrollada por Facebook, que se ha convertido en el estándar de la industria para el desarrollo de interfaces de usuario interactivas.

Lección 2: Componentes, JSX y Props

El corazón de React son los **componentes**. Un componente es una pieza de código independiente y reutilizable que representa una parte de la interfaz de usuario. Piensa en ellos como bloques de Lego que puedes combinar para construir una página.

**JSX** es una extensión de JavaScript que te permite escribir HTML directamente en tu código JavaScript. No es HTML puro, pero es muy similar y facilita la creación de componentes.

Las **Props** son la forma en que los datos se pasan de un componente padre a un componente hijo. Son como los "argumentos" o "propiedades" que le das a un componente para que sepa cómo debe renderizarse.

Lección 3: Estado (State) y Manejo de Eventos

El **Estado (State)** es una característica que permite que un componente tenga memoria y pueda cambiar su contenido a lo largo del tiempo. Por ejemplo, en un contador, el número actual es el estado. Cuando el usuario hace clic en un botón, el estado cambia y React actualiza la interfaz automáticamente.

El **Manejo de Eventos** te permite responder a las interacciones del usuario, como clics en botones o entradas de texto. Esto es lo que hace que tu aplicación sea verdaderamente interactiva y dinámica. 


                import React, { useState } from 'react';

                function Contador() {
                    const [conteo, setConteo] = useState(0);

                    return (
                        <div>
                            <p>El conteo es: {conteo}</p>
                            <button onClick={() => setConteo(conteo + 1)}>
                                Incrementar
                            </button>
                        </div>
                    );
                }
