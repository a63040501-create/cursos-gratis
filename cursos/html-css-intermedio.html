Curso: HTML y CSS Intermedio

Lección 1: El porqué de Flexbox y Grid

En el diseño web moderno, necesitamos herramientas que nos permitan crear diseños complejos y que se adapten a diferentes tamaños de pantalla (diseño responsivo). Anteriormente, los diseñadores web usaban trucos complejos con floats, pero hoy en día contamos con dos herramientas poderosas: **Flexbox** y **CSS Grid**. Estas herramientas son nativas del navegador y resuelven los problemas de diseño de forma mucho más eficiente.

  • **Flexbox** es ideal para alinear elementos en una **sola dimensión** (una fila o una columna). Útil para barras de navegación, tarjetas, etc.
  • **CSS Grid** es ideal para crear diseños en **dos dimensiones** (filas y columnas) al mismo tiempo, como el diseño de una revista o un tablero.

Lección 2: Fundamentos de Flexbox

Para usar Flexbox, debes crear un **contenedor padre** (flex container) y sus **elementos hijos** (flex items). El contenedor padre es el que tiene la propiedad `display: flex`. A partir de ahí, puedes alinear los elementos, justificar el espacio entre ellos y controlar su orden.

**Propiedades Clave del Contenedor Flexbox:**

  • `display: flex;` - Define el contenedor como un flexbox.
  • `flex-direction:` - Establece la dirección de los elementos (fila o columna).
  • `justify-content:` - Alinea los elementos a lo largo del eje principal (horizontal).
  • `align-items:` - Alinea los elementos a lo largo del eje secundario (vertical).

                .contenedor {
                    display: flex;
                    justify-content: center; /* Centra los elementos horizontalmente */
                }

Lección 3: Fundamentos de CSS Grid

CSS Grid te permite crear un diseño de cuadrícula en dos dimensiones. Puedes dividir tu página en filas y columnas y colocar los elementos en celdas específicas, lo que te da un control total sobre el diseño de la página.

**Propiedades Clave del Contenedor Grid:**

  • `display: grid;` - Define el contenedor como una cuadrícula.
  • `grid-template-columns:` - Define el número y el ancho de las columnas.
  • `grid-template-rows:` - Define el número y la altura de las filas.
  • `grid-gap:` - Crea un espacio entre las filas y las columnas.

                .contenedor {
                    display: grid;
                    grid-template-columns: 1fr 1fr 1fr; /* Tres columnas de igual tamaño */
                    grid-gap: 20px;
                }
