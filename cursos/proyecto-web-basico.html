Curso: Desarrollo de un Proyecto Web Básico

Lección 1: Planificación y Estructura del Proyecto

Antes de escribir una línea de código, el primer paso es la **planificación**. Define el objetivo de tu proyecto, su público objetivo y las funcionalidades que tendrá. Para este curso, construiremos una página de inicio simple con un botón interactivo.

La estructura de archivos es crucial para la organización. Vamos a crear una estructura de carpetas limpia y profesional en VS Code:

  • `mi-proyecto/`
    • `index.html` (La página de inicio)
    • `styles.css` (Para los estilos)
    • `main.js` (Para la interactividad)
    • `images/` (Para las imágenes)

Lección 2: Construyendo la Interfaz con HTML y CSS

Usaremos **HTML** para construir el esqueleto de nuestra página. Agregaremos un encabezado, un párrafo y un botón. Usaremos etiquetas semánticas para una buena práctica.

Luego, usaremos **CSS** para dar estilo a nuestra página. Le daremos colores, fuentes y un diseño atractivo. También usaremos Flexbox para centrar elementos, demostrando cómo lo aprendido en el curso intermedio es útil en la práctica. 


                .contenedor-principal {
                    display: flex;
                    justify-content: center;
                    align-items: center;
                    height: 100vh;
                }

Lección 3: Añadiendo Interactividad con JavaScript

Finalmente, usaremos **JavaScript** para añadir interactividad. Haremos que al hacer clic en un botón, un texto cambie o un elemento aparezca. Conectaremos nuestro archivo `main.js` al `index.html` y escribiremos el código para manipular el **DOM**.

**Ejemplo de código en `main.js`:** 


                const boton = document.getElementById('boton');
                const texto = document.getElementById('texto-oculto');
                
                boton.addEventListener('click', () => {
                    texto.textContent = '¡El texto ha cambiado!';
                });

Este proyecto te mostrará cómo las tres tecnologías (HTML, CSS y JavaScript) trabajan juntas para crear sitios web dinámicos.
