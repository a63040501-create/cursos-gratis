Curso: Desarrollo Web con Django

Lección 1: ¿Qué es Django?

**Django** es un *framework* web de alto nivel de Python que promueve el desarrollo rápido y un diseño limpio y pragmático. A diferencia de Flask, que es minimalista, Django viene con todo lo que necesitas para construir una aplicación web completa: un ORM para bases de datos, un sistema de autenticación de usuarios y un panel de administración. Se le conoce como un *framework* "de baterías incluidas".

El lema de Django es "El *framework* para perfeccionistas con plazos de entrega". Es ideal para construir sitios web complejos como redes sociales, tiendas en línea o plataformas de blogs.

Lección 2: El Modelo MTV (Model, Template, View)

Django se basa en el patrón de diseño **MTV (Modelo, Plantilla, Vista)**. Es similar a otros patrones populares como MVC.

  • **Modelo (Model):** Define la estructura de tus datos y se comunica con la base de datos. Si quieres almacenar un usuario, el modelo de usuario define sus campos (nombre, correo, contraseña).
  • **Plantilla (Template):** Es el archivo HTML que muestra la información al usuario. Es la capa de presentación.
  • **Vista (View):** Es la "lógica de negocio". Recibe una solicitud, interactúa con el modelo (si es necesario) y renderiza la plantilla. Es el puente entre el modelo y la plantilla.

Lección 3: Creando una Aplicación de Blog

Un proyecto común para aprender Django es una aplicación de blog. Los pasos básicos para crearla son:

  1. Crear un proyecto y una aplicación.
  2. Definir un modelo de `Post` con campos como título, autor y contenido.
  3. Configurar las URLs para que los usuarios puedan ver la lista de posts y un post individual.
  4. Crear las vistas que obtienen los datos del modelo.
  5. Crear las plantillas HTML para mostrar los posts.

Django se encargará automáticamente de la conexión con la base de datos y de la gestión de usuarios, permitiéndote enfocarte en la funcionalidad de tu blog.
