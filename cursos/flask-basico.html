Curso: Introducción a los Frameworks Web con Flask

Lección 1: ¿Qué es un *Framework* Web?

En los cursos anteriores, aprendiste a crear páginas web estáticas con HTML y CSS. Un **framework web** es un conjunto de herramientas y librerías que te permiten crear sitios web dinámicos. Un *framework* se encarga de las tareas repetitivas, como el manejo de las URL o el acceso a bases de datos, para que tú te enfoques en la lógica de tu aplicación.

**Flask** es un *micro-framework* de Python. Su principal ventaja es que es ligero y fácil de entender, lo que lo hace perfecto para tu primer acercamiento al desarrollo web con un lenguaje de *backend*.

Lección 2: Creando tu Primera Aplicación Flask

Para empezar a usar Flask, primero necesitas instalarlo en tu entorno de Python. Puedes hacerlo usando el siguiente comando en la terminal: 


                pip install Flask

Una vez instalado, puedes crear una aplicación web muy simple con solo unas pocas líneas de código. 


                from flask import Flask

                app = Flask(__name__)

                @app.route('/')
                def inicio():
                    return "¡Hola, mundo desde Flask!"

                if __name__ == '__main__':
                    app.run(debug=True)

Aquí, la línea `@app.route('/')` le dice a Flask que cuando el usuario visite la página principal (`/`), debe ejecutar la función `inicio()` y mostrar el mensaje.

Lección 3: HTML Dinámico con Plantillas

Una de las mayores ventajas de Flask es que te permite servir archivos HTML y llenarlos con información dinámica. Esto se logra usando **plantillas**. Flask usa el motor de plantillas **Jinja**, que te permite incrustar código de Python directamente en tu HTML.

Para usar plantillas, debes crear una carpeta llamada `templates/` en el mismo nivel que tu archivo de Python. 


                # Código de Python
                from flask import Flask, render_template

                app = Flask(__name__)

                @app.route('/')
                def inicio():
                    nombre = "Juan"
                    return render_template('index.html', nombre=nombre)
            

                <!-- Código de la plantilla index.html -->
                <h1>Hola, {{ nombre }}</h1>
