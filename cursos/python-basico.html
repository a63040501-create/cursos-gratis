Curso: Python Básico con VS Code

Lección 1: La Lógica de Programación y el Lenguaje Python

La programación es, en esencia, la habilidad de darle instrucciones a una computadora para que resuelva un problema. Antes de escribir cualquier código, debes pensar en la **lógica**: ¿cuáles son los pasos necesarios para llegar a la solución? Python es un lenguaje de programación de alto nivel que se caracteriza por su sintaxis limpia y legible, lo que lo hace perfecto para enfocarte en la lógica sin preocuparte por demasiadas reglas complejas.

Los **comentarios** (`#`) son esenciales. Sirven para documentar tu código y explicar lo que hace, lo que es vital para cuando regreses a él en el futuro o para cuando otros programadores lo lean.

Lección 2: Variables y Tipos de Datos Fundamentales

Una **variable** es un espacio en la memoria de la computadora que almacena un valor. Puedes pensarlo como una "caja" con una etiqueta que contiene información. El **tipo de dato** es el tipo de información que hay en esa caja (texto, números, etc.).

  • **Enteros (`int`)**: Números enteros (sin decimales), como `10` o `-5`.
  • **Flotantes (`float`)**: Números con decimales, como `3.14`.
  • **Cadenas (`str`)**: Texto, siempre va entre comillas (`"Hola Mundo"`).
  • **Booleanos (`bool`)**: Representan valores de verdad, con solo dos opciones: `True` o `False`.

Lección 3: Control de Flujo: Condicionales y Bucles

El **control de flujo** es lo que le da "vida" a tu programa, permitiéndole tomar decisiones y repetir acciones.

**Condicionales (`if`, `elif`, `else`)**: Son la forma en que los programas toman decisiones. Si una condición es verdadera, se ejecuta un bloque de código; si no, se ejecuta otro. Por ejemplo: 


                edad = 18
                if edad >= 18:
                    print("Eres mayor de edad")
                else:
                    print("Eres menor de edad")

**Bucles (`for`, `while`)**: Sirven para automatizar tareas repetitivas. El bucle `for` se usa cuando sabes la cantidad de repeticiones (ej. "recorre cada elemento en esta lista"), mientras que el bucle `while` se usa cuando la repetición depende de una condición (ej. "sigue pidiendo datos **mientras** la variable esté vacía").
