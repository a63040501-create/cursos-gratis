Curso: Introducción a JavaScript

Lección 1: ¿Qué es JavaScript?

JavaScript es el lenguaje de programación que le da **interactividad** a las páginas web. Si HTML es el esqueleto y CSS es la piel y la ropa, JavaScript es el **cerebro** que le dice a la página qué hacer. Permite crear efectos dinámicos, validar formularios, mostrar u ocultar elementos y comunicarse con servidores.

Es un lenguaje de programación interpretado, lo que significa que el navegador puede leer y ejecutar el código directamente. Esto lo hace muy rápido y eficiente para aplicaciones web.

Lección 2: Variables y Tipos de Datos

Al igual que en otros lenguajes, en JavaScript las **variables** son contenedores para almacenar datos. Puedes declararlas con las palabras clave `var`, `let` o `const`. 


                // Una variable que puede cambiar
                let nombre = "Juan";
                nombre = "Pedro";
                
                // Una constante, su valor no puede cambiar
                const PI = 3.1416;

                // Un tipo de dato de texto
                let saludo = "Hola, mundo";
                // Un tipo de dato numérico
                let edad = 30;
                // Un tipo de dato booleano
                let esActivo = true;

Aunque JavaScript es flexible con los tipos de datos, entender si estás trabajando con texto, números o valores lógicos es fundamental para evitar errores.

Lección 3: Manipulación del DOM

El **DOM (Document Object Model)** es una representación de la estructura de tu página web. JavaScript puede acceder a este modelo y manipularlo para cambiar el contenido, los estilos o la estructura de tu página. Es lo que permite, por ejemplo, que al hacer clic en un botón, aparezca un texto.

Puedes seleccionar elementos HTML usando métodos como `getElementById()`, `querySelector()`, o `getElementsByClassName()`. Una vez que tienes el elemento, puedes cambiar su contenido o sus estilos. 


                // Selecciona el elemento por su ID
                let titulo = document.getElementById("miTitulo");
                
                // Cambia el texto del elemento
                titulo.textContent = "Nuevo Título";
                
                // Cambia el color del texto
                titulo.style.color = "red";

La **manipulación del DOM** es el corazón de la interactividad en la web.
