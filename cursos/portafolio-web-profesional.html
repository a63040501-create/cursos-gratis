Curso: Desarrollo de un Portafolio Web Profesional

Lección 1: Planificación y Diseño (HTML Semántico)

Un portafolio web es tu tarjeta de presentación profesional. Antes de codificar, es vital planificar su estructura y contenido. Debe incluir secciones como: **Sobre Mí**, **Proyectos**, y **Contacto**.

Usaremos **HTML semántico** para construir una estructura sólida. Esto no solo hace que tu código sea legible, sino que también es crucial para la accesibilidad y para que los motores de búsqueda entiendan tu sitio. 


                <!DOCTYPE html>
                <html lang="es">
                <head>
                    <!-- Metadatos y CSS -->
                </head>
                <body>
                    <header>...</header>
                    <main>
                        <section id="sobre-mi">...</section>
                        <section id="proyectos">...</section>
                        <section id="contacto">...</section>
                    </main>
                    <footer>...</footer>
                </body>
                </html>

Lección 2: Estilo y Diseño (CSS Intermedio)

Aquí es donde aplicarás lo aprendido en el curso de HTML y CSS Intermedio. Utilizaremos **Flexbox** para centrar elementos y crear diseños de una sola fila, y **CSS Grid** para organizar las secciones de proyectos en una cuadrícula atractiva.

También te enseñaremos a crear una paleta de colores profesional y a elegir tipografías que mejoren la legibilidad y la apariencia de tu portafolio.

Lección 3: Publicación del Portafolio con Git y GitHub Pages

El último paso es publicar tu portafolio para que todo el mundo pueda verlo. Utilizaremos **Git y GitHub Pages** para lograrlo, lo que te permite alojar tu sitio web de forma gratuita.

Aplicarás los conocimientos del curso de Git para inicializar un repositorio, hacer commits y subir tu código a GitHub. Luego, con solo un par de clics, activarás GitHub Pages para que tu portafolio esté en vivo en Internet. Esto es crucial para que los reclutadores y las empresas puedan ver tu trabajo.
