Curso: Introducción a la Ciencia de Datos con Python

Lección 1: ¿Qué es la Ciencia de Datos?

La **Ciencia de Datos** es la disciplina que se encarga de extraer conocimiento y valor de los datos. No se trata solo de números, sino de encontrar patrones, tendencias y "historias" que nos ayuden a tomar mejores decisiones. Utiliza una combinación de programación, matemáticas y conocimiento del negocio.

Un científico de datos, por ejemplo, podría analizar datos de ventas para predecir qué productos serán los más populares el próximo trimestre.

Lección 2: Análisis de Datos con Pandas

**Pandas** es la librería más utilizada en Python para la manipulación y el análisis de datos. Utiliza una estructura de datos llamada DataFrame, que es similar a una hoja de cálculo, lo que la hace muy intuitiva para trabajar.

Verás cómo usar Pandas para cargar datos de archivos CSV, limpiar información desordenada, filtrar datos y calcular estadísticas. 


                import pandas as pd

                # Carga un archivo CSV en un DataFrame
                df = pd.read_csv('ventas.csv')

                # Muestra las primeras 5 filas
                print(df.head())

                # Calcula el promedio de la columna 'precios'
                print(df['precios'].mean())

Lección 3: Aprendizaje Automático Básico

El **Aprendizaje Automático (Machine Learning)** es una rama de la inteligencia artificial que permite a las computadoras aprender a partir de los datos sin ser programadas explícitamente.

Usaremos la librería **Scikit-learn** para construir un modelo simple. Por ejemplo, podríamos crear un modelo de regresión lineal que prediga el precio de una casa basándose en su tamaño y ubicación. Este es el primer paso para crear sistemas inteligentes.
