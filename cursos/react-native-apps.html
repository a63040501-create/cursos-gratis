Curso: Desarrollo de Apps con React Native

Lección 1: ¿Qué es React Native?

React Native es un framework de JavaScript que te permite construir aplicaciones móviles nativas usando los mismos principios de React. A diferencia del desarrollo web, donde el código se ejecuta en un navegador, una app de React Native se compila en código nativo de iOS y Android. Esto significa que la app tiene el rendimiento y la apariencia de una aplicación nativa.

La principal ventaja es la filosofía de "escribe una vez, ejecuta en cualquier lugar", lo que te permite usar una base de código para dos plataformas móviles.

Lección 2: Componentes Nativos y Estilos

En lugar de usar etiquetas HTML como <div> y <p>, React Native usa componentes nativos que se traducen en elementos de la interfaz de usuario de iOS o Android. Por ejemplo, en lugar de <div>, usas el componente <View>, y en lugar de <p>, usas <Text>.

El estilo en React Native se hace con JavaScript, usando una sintaxis similar a CSS, pero sin los selectores de cascada. Esto te da un control preciso sobre cómo se ve cada componente. 


                import { Text, View, StyleSheet } from 'react-native';

                const styles = StyleSheet.create({
                    container: {
                        flex: 1,
                        justifyContent: 'center',
                        alignItems: 'center',
                    },
                    texto: {
                        fontSize: 20,
                        color: 'blue',
                    }
                });

Lección 3: Navegación y Acceso al Dispositivo

Aprenderás a usar librerías como React Navigation para manejar la navegación entre pantallas. También verás cómo acceder a las funcionalidades del dispositivo, como la cámara, el GPS o los sensores, usando librerías especializadas que simplifican el proceso.

Estas habilidades son esenciales para crear aplicaciones móviles completas que se sienten y funcionan como apps nativas.
