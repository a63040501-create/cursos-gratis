Curso: HTML y CSS Básico

Lección 1: HTML, la Estructura y la Semántica

HTML (HyperText Markup Language) es el **lenguaje de marcado** que le da estructura a una página web. Piensa en HTML como los cimientos de una casa. Sus "ladrillos" son las **etiquetas** (`

`, `

`, ``), que envuelven el contenido para darle un significado.

El **HTML semántico** es un concepto fundamental. Consiste en usar las etiquetas de la forma correcta para que el contenido tenga un significado claro. Por ejemplo, en lugar de usar una etiqueta `

` para todo, usamos etiquetas que describen su propósito, como `
`, `
`, `
`, y `
`. Esto es crucial para la **accesibilidad** y los motores de búsqueda.

Lección 2: CSS, el Estilo y el Modelo de Caja

Si HTML es la estructura, **CSS (Cascading Style Sheets)** es el estilo, el diseño y los colores. CSS le dice al navegador cómo debe verse el contenido de HTML, permitiéndote cambiar fuentes, colores, y la posición de los elementos.

El **Modelo de Caja (Box Model)** es un concepto central en CSS. Cada elemento HTML es una caja, y CSS te permite controlar sus propiedades:

  • **Contenido**: El espacio para el texto o la imagen.
  • **Padding**: El "relleno", el espacio interno entre el contenido y el borde.
  • **Border**: La línea que rodea la caja.
  • **Margin**: El "margen" externo, que crea espacio entre cajas.

Puedes seleccionar elementos para darles estilo usando **selectores**, que apuntan a etiquetas (`p`), clases (`.clase`) o IDs (`#id`).
