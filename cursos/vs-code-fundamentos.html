Curso: Fundamentos de VS Code

Lección 1: ¿Por qué VS Code? Un Entorno de Desarrollo Integrado (IDE)

Visual Studio Code es mucho más que un simple editor de texto como el Bloc de notas. Es lo que llamamos un **IDE (Entorno de Desarrollo Integrado)**. Esto significa que está diseñado específicamente para programar, ofreciéndote un conjunto de herramientas esenciales en un solo lugar: un editor de código, una terminal, un depurador y soporte para control de versiones como Git.

La **Barra de Actividad** en el lado izquierdo te da acceso a las vistas principales: el Explorador de Archivos, la Búsqueda, el Control de Código Fuente (Git), la ejecución y depuración de código, y las extensiones.

Lección 2: Extensiones, Atajos y el Flujo de Trabajo

Las **extensiones** son el mayor superpoder de VS Code. Piensa en ellas como "aplicaciones" que puedes instalar para añadir funcionalidades específicas. Las extensiones de lenguajes te dan autocompletado inteligente y la capacidad de ejecutar código, mientras que otras como **Live Server** te permiten ver los cambios en tus páginas web en tiempo real.

Aprender **atajos de teclado** es clave para ser un programador eficiente. Uno de los más importantes es **`Ctrl + Shift + P`** (en Windows/Linux) o **`Cmd + Shift + P`** (en Mac). Este atajo abre la **Paleta de Comandos**, que te permite acceder a todas las funciones de VS Code sin tener que buscar en los menús.
